# Nodify PHP Client

Official Nodify API client for PHP. Nodify is a powerful content management system that enables dynamic content delivery with built-in translation and personalization features.

## 🚀 Features

- ✅ Full API coverage (Nodes, Content Nodes, Users, Plugins, etc.)
- ✅ Authentication with JWT tokens
- ✅ Automatic token management
- ✅ PSR-4 autoloading
- ✅ Guzzle HTTP client
- ✅ Comprehensive error handling
- ✅ Support for all Nodify features:
    - 📁 Node management (Sites, Pages, Containers)
    - 📄 Content nodes with translations
    - 🌍 Multi-language support with `$translate` directive
    - 🎯 Dynamic content with `$value` directive
    - 🔒 Lock system
    - 📊 Analytics
    - 🔌 Plugins

## 📦 Installation

### Using Composer (recommended)

```bash
composer require nodify/php-client

# Nodify News Feed Example - Python Client

A complete example of a dynamic news feed built with the Nodify Python client. This example demonstrates:

- Creating a hierarchical content structure with categories and articles
- Multi-language support (EN, FR, ES) using `$translate()` directives
- Filtering articles by category
- Adding and retrieving comments (using the Data API)
- Publishing content and consuming it via API

## Prerequisites

- Python 3.8+
- Nodify CMS instance running (see [Nodify documentation](https://github.com/AZIRARM/nodify-clients))
